**Design Notes:**
- Data-focused layout
- Export functionality prominent
- Field mapping accessible
- Consistent with other pages

## Design System Components

### Buttons
- Primary: Solid, prominent
- Secondary: Outlined
- Ghost: Text only
- Danger: Red variant
- Sizes: sm, md, lg

### Cards
- Default: White background, subtle shadow
- Elevated: More shadow for emphasis
- Interactive: Hover effects
- Bordered: Outlined variant

### Inputs
- Default: Clean border
- Focus: Blue accent
- Error: Red border + message
- Success: Green border
- Disabled: Grayed out

### Tables
- Clean rows
- Hover states
- Sortable headers
- Action buttons per row
- Pagination

### Modals/Panels
- Side panel (preferred for detail views)
- Centered modal (for confirmations)
- Backdrop overlay
- Close button
- Smooth animations

## Layout Grid System

### Desktop (1200px+)
- 12-column grid
- Container: max-width 1600px
- Sidebar: 280px (collapsible)
- Main: Remaining space

### Tablet (768px - 1199px)
- 8-column grid
- Sidebar: Collapsible or hidden
- Main: Full width

### Mobile (< 768px)
- Stack layout
- Bottom navigation
- Full-width cards
- Touch-optimized
